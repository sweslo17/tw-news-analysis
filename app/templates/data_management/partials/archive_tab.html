Archive raw HTML to compressed files and free up database space. Uses additive mode - existing archives are preserved.

Restore Raw HTML

Restore archived raw HTML back to the database.

About Archiving

  • Archives are stored as compressed JSON files (gzip)
  • Each batch contains up to 500 articles
  • Archiving uses additive mode - new archives don't affect existing ones
  • Auto-archive runs daily at 01:00 for yesterday's data
